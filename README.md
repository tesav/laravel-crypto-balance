# Тестовое задание: Модуль учета крипто-баланса пользователя

## Описание

Цель задания — разработать модуль, который позволяет корректно и безопасно управлять крипто-балансом пользователя.  
Модуль должен обеспечивать:

- **Зачисление средств** на баланс пользователя.
- **Списание средств** (выводы, платежи, комиссии).
- Учет специфики блокчейна, асинхронности и рисков.

Реализация выполнена на **PHP + Laravel**.

1. Склонируйте репозиторий:
